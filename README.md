# Tidewater

A small open-world **coastal-town slice** you stroll through, built in **Godot 4.6.3** and exported to
the web (mobile-first, runs in Safari / Chrome / Firefox on phone and desktop).

Wander a sandy beach that slopes into an animated, foamy ocean; past 2–3 blocks of pastel seaside
buildings, palms and streetlights; talk to a small wandering crowd of townsfolk; and watch the town
move through a **day → sunset → night → sunrise** cycle that lights the streets after dark. Stroll east
to the old lighthouse on the point.

It's a calm, no-combat experience — just exploration and conversation.

## Controls
- **Move:** on-screen left-half drag (touch) or **WASD / arrow keys**.
- **Look:** on-screen right-half drag (touch) or **left-click drag** (desktop) to orbit the camera.
- **Talk:** walk up to a townsperson and press **USE**.

## How it's built
- **Chunk-streamed open world** (`world.json`, `mode:"chunk"`): a 6×5 grid of 16 m cells around the
  player, streamed in/out with no loading screens. The beach/ocean, buildings, palms, streetlights,
  crowd and townsfolk are all authored as data in `world.json` (generated by `gen_world.py`).
- **Terrain + ocean:** a noise heightfield with a **coastal ramp** (`terrain.gd`) so the land descends
  through the beach into the sea, and the toon-water ocean (`water.gd`/`water.gdshader`) foams along
  the shoreline derived from that terrain.
- **Buildings + lighthouse:** parametric structures (`build_structure.gd`) with lit windows; a tapered
  white tower + beacon for the lighthouse.
- **Night lighting:** per-cell `lamps` — a pole + an emissive head + a warm `OmniLight` pool — so the
  town lights itself once the sky darkens.
- **Characters:** rigged, animated (idle/walk) characters generated with **Meshy AI** — the player
  avatar plus the townsfolk and the wandering crowd. The bulk of the props (palms, docks, shells,
  benches) come from the curated CC0 asset library and **stream from R2 at runtime**.
- **Talking:** townsfolk reply in-character through the shared NPC brain (`npc.myapping.com`).
- **Audio:** localized, distance-faded ocean waves at the shore, crowd murmur around the crowd, and
  footsteps on the player, over a calm town track + a quiet sea breeze.

## Building
Binary assets (`.glb` models, `.ogg` audio) are not committed — fetch them, then export:

```bash
./scripts/fetch_assets.sh                      # pulls the packed player/audio assets
python3 gen_world.py                           # regenerates world.json (the world is data, generated here)
# install Godot 4.6.3 (Compatibility/WebGL2, nothreads web export templates), then:
godot --headless --import
godot --headless --export-release "Web" out/index.html
cp world.json quests.json out/                 # loose data files served next to index.html
```

Then serve `out/` over HTTP (the web export needs correct `.wasm` MIME and no COOP/COEP headers).

No backend is required — Tidewater is single-player and fully client-side.
